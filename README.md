## Getting Started

### Prerequisites

This project requires the following dependencies:

* **Programming Language**: PHP
* **Package Manager**: Composer

---

### Installation

Build **Bat_Cave_Cafe** from the source and install dependencies:

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/Bat_Cave_Cafe
   ```

2. **Navigate to the project directory**

   ```bash
   cd Bat_Cave_Cafe
   ```

3. **Install the dependencies**

   ```bash
   composer install
   ```

---

### Usage

Run the project using:

```bash
php -S localhost:8000
```

Access the application via your browser to manage menus, reservations, and content.

---

### Testing

Bat_Cave_Cafe uses the **Pest** testing framework. Run the test suite with:

```bash
vendor/bin/pest
```
